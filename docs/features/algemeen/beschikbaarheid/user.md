# Beschikbaarheid (Gebruiker)

## Doel van de module
Het doel van de Beschikbaarheid module is om medewerkers van **Apotheek Jansen** de mogelijkheid te bieden hun beschikbaarheid voor werk door te geven. Dit stelt de planners in staat om efficiënt een rooster op te stellen en vermindert de noodzaak voor afstemming via externe kanalen zoals WhatsApp.

## Wat kun je met deze module?
Met deze module kun je:

- Per dag aangeven of je beschikbaar bent voor werk.
- Beschikbaarheid doorgeven voor specifieke dagdelen (bijv. Ochtend, Middag, Vooravond).
- Je eigen doorgegeven beschikbaarheid inzien en wijzigen voor toekomstige periodes.
- Zien welke beschikbaarheid automatisch is gegenereerd (voor medewerkers met een vast contract).

## Werkwijze
Je kunt je beschikbaarheid doorgeven via het menu "Mijn Beschikbaarheid".

1. Open de Beschikbaarheid module vanuit het hoofdmenu.
2. Navigeer naar de gewenste maand of week.
3. Klik op een dag om de dagdelen te selecteren waarop je beschikbaar bent.
4. Sla je wijzigingen op om de planning op de hoogte te brengen.

## Bijzonderheden
- Voor medewerkers met een vast dienstverband wordt de beschikbaarheid automatisch ingevuld op basis van hun contractuele afspraken in hun profiel.
- Handmatig doorgegeven beschikbaarheid overschrijft in veel gevallen de automatische instellingen voor die specifieke dag.
- Er wordt onderscheid gemaakt tussen de bron van beschikbaarheid: "Auto" (gebaseerd op contract) of "Manual" (handmatig doorgegeven door de gebruiker).
