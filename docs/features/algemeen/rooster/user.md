# Rooster (Gebruiker)

## Doel van de module
Het doel van de Rooster module is om medewerkers van de **Apotheek Jansen App** inzicht te geven in de actuele werkplanning en bezetting binnen de organisatie.

## Wat kun je met deze module?
Met deze module kun je:

- Het actuele weekrooster inzien via de applicatie.
- Terugkijken naar roosters van voorgaande weken.
- Vooruitkijken naar toekomstige roosters (mits gepubliceerd).
- Een gedetailleerde weergave van het rooster bekijken in de vorm van pagina's of afbeeldingen.
- Directe meldingen ontvangen wanneer er een nieuw rooster beschikbaar is gesteld.

## Werkwijze
Het rooster is een centraal onderdeel van de applicatie en is vaak direct zichtbaar op het dashboard of via het menu "Rooster".

1. Open de Rooster module vanuit het hoofdmenu.
2. Bekijk standaard het rooster van de huidige week.
3. Gebruik de navigatieknoppen om naar een specifieke week te gaan.
4. Klik op de rooster-weergave om een vergroting of de volledige PDF te bekijken.

## Bijzonderheden
- Roosters worden vaak geüpload als PDF-bestand en door het systeem automatisch omgezet naar afbeeldingen (PNG/WebP) voor een optimale weergave op mobiele apparaten.
- Er wordt per week een overzicht bijgehouden, waarbij de maandag van de betreffende week als startpunt dient.
- Zodra een nieuw rooster is geüpload, worden alle medewerkers hiervan op de hoogte gesteld via een push-notificatie.
