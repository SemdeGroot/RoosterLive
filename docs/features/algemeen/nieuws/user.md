# Nieuws (Gebruiker)

## Doel van de module
Het doel van de Nieuws module is om medewerkers van de **Apotheek Jansen App** op de hoogte te houden van belangrijke berichten, aankondigingen en updates binnen de organisatie.

## Wat kun je met deze module?
Met deze module kun je:

- Belangrijke nieuwsberichten en aankondigingen inzien via de applicatie.
- PDF-documenten en afbeeldingen bekijken die bij een nieuwsbericht horen.
- Een korte beschrijving van het nieuwsbericht lezen voordat je het volledige bericht opent.
- Directe meldingen ontvangen wanneer er een nieuw bericht is geplaatst.

## Werkwijze
Het nieuws is direct toegankelijk via het menu "Nieuws" of via de dashboardtegel in de applicatie.

1. Open de Nieuws module vanuit het hoofdmenu.
2. Bekijk de lijst met nieuwsberichten gesorteerd op datum (meest recente bovenaan).
3. Klik op een bericht voor de details en eventuele bijlagen.
4. Voor PDF-bijlagen wordt er vaak een preview getoond in de browser.

## Bijzonderheden
- Nieuwsberichten hebben een automatische houdbaarheid. Berichten ouder dan 3 maanden worden periodiek opgeschoond om de lijst relevant te houden.
- Beheerders kunnen nieuwe berichten plaatsen inclusief PDF- of afbeeldingsbestanden.
- Bij het uploaden van een nieuw bericht wordt er een push-notificatie verstuurd naar alle medewerkers.
