# Welkom bij de Documentatie

Welkom bij de interne documentatie van de Apotheek Jansen App. Dit platform ondersteunt onze apothekers en zorgverleners bij hun dagelijkse werkzaamheden, met de nadruk op de medicatiebeoordeling.

Hier vind je informatie over de werking van de app. Zowel voor de dagelijkse gebruiker (uitvoeren van reviews) als voor beheerders (aanpassen van beslisregels en technische achtergrond).

Gebruik het menu om door de onderwerpen te navigeren.
